Sistema de Gestão Hospitalar

Caio Lopes e Luisa Longo

Funcionamento do Projeto

Nosso sistema é uma plataforma médica de gestão desenvolvida em C++ com uma interface funcional projetada no Qt Creator. O sistema foi arquitetado seguindo os princípios da programação orientada a objetos, garantindo uma estrutura organizada, escalável e de fácil manutenção.
